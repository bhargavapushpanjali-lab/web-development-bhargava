How to Make Nested List?

Nested list means list inside list.

You can make any of these nested list in HTML.

  1. Ordered list inside Unordered list.
  2. Unordered list inside Ordered list.
  3. Ordered list inside Ordered list.
  4. Unordered list inside Unordered list.

How to make table in HTML?

  • We all know that in table there are rows and columns.
  • Rows are horizontal arrangement of cells whereas columns are vertical arrangement of cells.
  • In HTML we have to learn how to create row and column to form a table.
  • So whenever you want to create a table firstly you have to use table tag to declare that you are going to create a table.
  • To create the border of the table you have to write inside the table tag border="1".
  • To create or add the row in the table you have to use tr tag.
  • Inside the tr tag you have to use td tag to fill the data in the row.
  • To highlight the headings of table you have to use th tag inside the tr tag.
  • Inside the table tag it is recommended to use three tags which represent the three parts of the table.
    1. thead: Inside this tag we will write heading of the table using th tag.
    2. tbody: This tag contains body of the table.
    3. tfoot: This tag contains conclusion of the table.
  • To extend column we use colspan tag. While using this tag we have to give the value of colspan according to this that how many columns it is going to cover.If it is going to cover 3 columns then the value of colspan is equal to 3.
  • Similarly to extend row we use rowspan tag. While using this tag we have to give the value of rowspan according to this that how many rows it is going to cover.If it is going to cover 2 rows then the value of rowspan is equal to 2.
  • You can also use both rowspan and colspan together as well.
