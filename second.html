What Is HTML?

  • Full form of HTML is Hyper Text Markup Language.
  • It creates static website only contains content.
  • It creates website which contains hyper link which helps us to navigate from one page to another.
  • .html is the extension for html file.

Why We Need To Use Tags In HTML?

If we don't use tags in HTML then it will show our whole content just like article without any heading and without any gap between paragraphs.

That's why it is essential to use tags to provide proper structure to our content.

Tags are used to provide semantic meaning and proper structure to the words that we have written in HTML file.

And this is what we meant by markup language.

TAGS

  1. Heading Tags:It gives semantic meaning to the content.

    There are 6 levels of heading.

    h1 Heading of web page.
    h2 Sub heading of h1
    h3 Sub Heading of h2
    h4 Sub heading of h3
    h5 Sub Heading of h4
    h6 Sub heading of h5

    These 6 heading deyermines the preference and priority order of topics in web page.

    • It is recommended to never jump heading. Preference order should be maintained.
    • In a web page only one h1 heading can be there.
  2. Paragraph Tag: This tag is used to write paragraph. This tag is represented by p inside < >.
  3. Line Break Tag: This tag is used to give more gape between words,sentences and paragraphs.This tag is represented by br inside < >.
  4. Horizontal Rule Tag: This tag is used to draw a horizontal line. represented by hr inside < >.
  5. Image Tag: This tag is used to add image in website. You have to give source by writing src="link of image" to add photo. You can also give it's height and width.

Difference between tag and element

A tag is part of the markup.Example p inside < >.

An element is the entire structure,including tags and content.

How to create Hyperlink in HTML?

For this purpose we use Anchor Tag which is represented by a inside < >. We need to type the link in href then the hyperlink will be created.

By default link will be open in that tab only.

To open link in new tab we need to write target="_blank".

Attributes

How to add image in website?

As we discussed above about img tag for adding image.By following the steps mentions below you can easily add the image in your website.

  1. Search for the image in web browser you want to add.
  2. Select that image.
  3. Right click on the image
  4. Select open image in new tab.
  5. Copy the link of image from that tab.
  6. Paste the link of image in src.
