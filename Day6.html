Media Tag

  • This tag is used to integrate audio,video and youtube video with web page.

    1. To integrate audio files

      • For this you have to use audio tag.
      • Determine the source of file with the help of src inside audio tag.
      • You also have to write controls inside audio tag to access the audio in your web page.

      Alternate Method

      This method is proper to integrate audio files.

      • Initiate with audio tag.
      • Inside it write controls
      • Then under audio tag determine it's source by src and also it's type

    2. To integrate audio files

      • For this you have to use video tag.
      • Determine the source of file with the help of src inside video tag.
      • You also have to write controls inside video tag to access the video in your web page.

      Alternate Method

      This method is proper to integrate video files.

      • Initiate with video tag.
      • Inside it write controls
      • Then under video tag determine it's source by src and also it's type

    3. To integrate youtube video

      • Search your video in youtube.
      • To integrate youtube video we use the iframe tag.
      • Also determines it's source using src.
      • You have to replace watch word in link with embed.
      • If you write full screen inside iframe tag you can access your video with full screen

How to create multipage website.

  • To create multipage website we use anchor tag inside ordered or unordered list.
  • This leads to the creation of hyerlinks of all the pages.
