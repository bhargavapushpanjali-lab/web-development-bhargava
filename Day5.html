What is form?

Form allow users to submit the entered data to a server for processing.

Steps to create the form

  1. Use form tag to declare that you are creating the form.
  2. Use input tag to create the cox in the form.
  3. Determine the type of input you will enter in the box. For this purpose you can use type="".
  4. Use label tag before input tag to tell the user what should be fill in the box.
  5. To create submit button, use input tag. Inside input tag declare it's type="submit".

The types of input we can use in HTML

  1. Text
  2. Number
  3. Password
  4. Submit
  5. Radio
  6. checkbox
  7. date
  8. email
  9. tel
  10. file


Now whenever we take the information of user it goes somewhere. For now we can say that this information goes to database.

In Database there is the excel sheet already formed by the backend developer in which the information of user will be stored.

But ever you thought that how it gets to know that which type of information has come and where to store it in excel sheet.

Here comes the concept of name-value concept

Name-value Attribute

  • As we discussed above that excel sheet does not get to know what type of information had came and wher to store it.
  • So we need to send complete data to the database that what type of information is that and what value it have to store.
  • So for the determination of which type of value we are sending to database we use name keyword.
  • We will add name keyword inside input tag.
  • Name keyword can be same as that of level and can be different also.
  • Value is equal to data which we have entered in the box of the form.

radio Buttons

  • We use type="radio" inside input tag when we need to select single option from multiple option of same name keyword.
  • It is used when only one value can be taken in name keyword

for keyword

  • You have noticed that sometime when we click on the label mentioned before the box it directly navigates us to inside the box.
  • This happens with the help of for keyword which is declared inside label tag.
  • We have to link for with id for which we should give same name to both of them.

value keyword

  • If you designed the buttons in your form then when you fill that form so when you click on that button it takes value as on and another click results the value as off.
  • To resolve this problem we have to use value keyword inside input tag and ensures that what value should the database take when we on/off the button.

checkboxes

  • By using checkboxes multiple values can be taken as input.
  • To create checkboxes in the form we have to write type="checkbox" inside input tag.

placeholder Attribute

  • This attribute is used when we want to add an example inside box that which type of data should be entered in box.
  • For this we have to write like this placeholder="eg:abc" inside the input tag.

textarea tag

  • It is used when we have to enter large text inside the box of the form.
  • It increases the area of box in the form.
  • It's size can be determine by entering the value of height and width.

select tag

  • It is used when we have to create dropdown option in the box.
  • With select tag you also have to use option tag to determine the options in dropdown.

button tag

  • It is used to create buttons in the form.

required Attribute

  • It is used to mark important fields in the form and without filling those fields form wil not be submitted.
  • For this you just have to write required inside input tag.
