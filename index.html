What is Web Development?

Before understanding web development we should understand what is internet and how it works.

So here the one more question arises that why we use internet?

The answer of this question is that to communicate with each other and to transfer data from one device to another we need internet.And to transfer data we need address so that we can send data to another one just like postal system.

So here the concept of IP address comes in role

What is IP Address?

When the device is connected to the internet it has given an address which is called IP address. And when it is disconnected from internet IP address IP address wil be taken from that device.

Things to remember about IP Address.

  1. The full form of IP address is Internet Protocol.
  2. It is provided by ISP(Internet Service Provider).
  3. It is not the permanent address of the device.
  4. It is a dynamic address.

The data is transefered in the form of the way described below:

Sender's IP Address Data Receiver's IP Address

So from the concept of IP address we have known that we need an address to transfer data from one device to another.

Ever you thought that when we search for any website in web browser we don't write the address of that server in which that website is present then how we get the web page in result.

So here comes the DNS(Domain Name System) for the rescue.

DNS (Domain Name System)

It is the system which maintains the book for the user of device.It is the book which contains IP address of each and every device.

For example:

The following steps happen when you search any website in your web browser.

google.om 13.46.5.4
coderarmy.in 50.16.11.21
  • Whenever we search any website in our browser.
  • The request is send to DNS system.
  • To ask for the IP address of server which contains the website.
  • DNS returns the IP address of server.
  • Then computer sends data to the server.
  • Server returns the data to computer and then website opens.

The constituents of DNS System.

  1. Recursive Server
  2. Root Server
  3. TLD (Top Level Domain)
  4. Authoritative Sever

So till now we have understood the meaning of IP address and role of DNS in our system.

Why we need Mac Address?

In our device we need Mac address because IP address is not permanent address while exchanging data from one computer to another if any one of them is disconnected frpm internet then in that case IP address has been taken from that computer and ISP can give that address to any other device results in miscommunication of data.

It becomes more dangerous when data is sensitive.So we need permanent address for our device that's why Mac address has been given to each device.

Points to be remember about Mac address:

  1. Mac address is permanent address as it comes physically attached with device.
  2. Mac address is in hexadecimal form.
  3. The number of Mac address of a device is equal to number of ways to transfer data from our device to another.

As we have discussed about Mac address so parallel to it let's understand the concept of Port number.

Port Number

By Port Number, device can determine in which tab data should be display after receiving the message.

So finally our message looks like this when we send message to another device.

Sender's IP address Sender's Mac address Sender's Port Number data Receiver's IP address Receiver's Mac address Receiver's Port Number

How Internet Works?

If we think of the methods to transfer data from one device to another we came across with these two ways:

  1. By using pendrive
  2. By connecting devices through cables.

But both have their respective drawbacks and impractical way to transfer data when we talk about tranfering data on large scale.

So here comes the role of LAN(Local Area Network),Router and Network

Networks

  • Internet helps to connect LAN all across the world.
  • Private IP address of a computer is unique only within the LAN.
  • But in different LAN private address can be same.
  • Public IP address is globally unique address.
  • Router has given Public IP address to stand out uniquely.
